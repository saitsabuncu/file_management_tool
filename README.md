# File Management Tool

Python öğrenme sürecimde dosya ve klasör işlemlerini, otomasyon mantığını ve dosya türlerine göre sınıflandırma işlemlerini uygulamalı olarak öğrenmek amacıyla geliştirdiğim basit bir dosya yönetim aracıdır.

## Proje Hakkında

Bu araç, kullanıcının belirlediği bir klasördeki dosyaları uzantılarına göre analiz eder ve dosyaları kategori klasörlerine taşır.

Dosyalar şu kategorilere ayrılır:

* Resimler
* Belgeler
* Videolar

## Özellikler

* Klasör içeriğini listeleme
* Dosyaları uzantılarına göre kategorilere ayırma
* Hedef klasörde kategori klasörleri oluşturma
* Dosyaları ilgili kategori klasörlerine taşıma
* Hatalı klasör yolu için temel hata kontrolü

## Kullanılan Teknolojiler

* Python 3
* os
* shutil

## Nasıl Çalıştırılır?

```bash
python main.py
```

Program çalıştırıldığında kullanıcıdan:

1. Kaynak klasör yolu
2. Hedef klasör yolu

bilgileri istenir.
